# Question Site

A web app made with Gin and Vue that allows users to ask and answer questions.

## Installation

```bash
git clone https://github.com/esgameco/question-site.git
cd questionclient
npm i
export SECRET="insert encryption key"
```

## Run

#### Development

```bash
Linux: ./devrun.sh
Windows: .\devrun.bat
```
